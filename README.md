## Getting Started [ğŸ”](#lendsqr-frontend-engineering-assessment)

### Prerequisites [ğŸ”](#lendsqr-frontend-engineering-assessment)

Before running the project, ensure you have the following installed:

- **Node.js (Latest LTS recommended)**

To update npm to the latest version, run:

```sh
$ npm install -g npm@latest
```

### Installation [ğŸ”](#lendsqr-frontend-engineering-assessment)

Clone the repository:

```sh
$ git clone https://github.com/topghostly/lendsqr-fe-test.git
$ cd lendsqr-fe-test
```

Install dependencies:

```sh
$ npm install
```

Run the development server:

```sh
$ npm run dev
```

Enter your Mocky API endpoint in `.env.local`:

```sh
$ const MOCK_API_UR = 'ENTER YOUR API ENDPOINT';
```

Visit `http://localhost:3000` in your browser.

### Mock API Setup [ğŸ”](#lendsqr-frontend-engineering-assessment)

The application uses a mock API with 500 records. You can generate mock data using:

- [Mocky.io](https://mocky.io/)

### Testing [ğŸ”](#lendsqr-frontend-engineering-assessment)

```sh

```
